class owlapy.agen_kg.chunking_models.simple_chunker.TextChunker(chunk_size: int = 3000, overlap: int = 200, strategy: str = 'sentence', enable_logging: bool = False)[source]

Utility class for splitting large texts into smaller chunks that fit within an LLM’s context window. Supports multiple chunking strategies.

This is essential for handling large documents (e.g., long PDFs, books, extensive reports) when using LLMs with limited context windows.
